詳解

動的にメモリを確保しながら書き込むストリームクラス 通常はSizeCaliculator → StreamWriterの流れで書き込むが、 ストリームに書き込んでいる最中に動的な処理を組んだりするケースでは こちらのクラスを使う事。 ※Serializeを実行するたびに、内部的にnewとdeleteを繰り返すので注意。
